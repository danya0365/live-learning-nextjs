## üóÉÔ∏è Database Schema

10 tables, PostgreSQL via Supabase

```
auth.users
  ‚îî‚îÄ‚îÄ profiles            # 1:N (multi-profile)
       ‚îú‚îÄ‚îÄ profile_roles   # 1:1 (student/instructor/admin)
       ‚îú‚îÄ‚îÄ instructor_profiles  # extends profile
       ‚îÇ   ‚îú‚îÄ‚îÄ courses
       ‚îÇ   ‚îú‚îÄ‚îÄ time_slots
       ‚îÇ   ‚îú‚îÄ‚îÄ live_sessions
       ‚îÇ   ‚îî‚îÄ‚îÄ consultation_offers
       ‚îú‚îÄ‚îÄ bookings (as student)
       ‚îî‚îÄ‚îÄ consultation_requests (as student)

categories
  ‚îú‚îÄ‚îÄ courses
  ‚îî‚îÄ‚îÄ consultation_requests
```

| Table | Purpose |
|-------|---------|
| `profiles` | User profiles (extends auth.users) |
| `profile_roles` | Role assignment (student / instructor / admin) |
| `categories` | Course categories (6 types) |
| `instructor_profiles` | Instructor data (bio, rating, hourly_rate) |
| `courses` | Course catalog |
| `time_slots` | Weekly instructor availability |
| `bookings` | Class reservations |
| `live_sessions` | Live streaming sessions |
| `consultation_requests` | Student help requests |
| `consultation_offers` | Instructor offers on requests |

### Prerequisites

